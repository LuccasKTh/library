## Rodar os comandos:
```
  npm install
```
```
  composer install
```

## Após instalar as dependências:
  1. Copiar e colar o arquivo .env.exemple e renomar para .env
  2. Atualizar os dados de conexão no .env
  3. Rodar os arquivos .sql na pasta config
  
## Rodar o comando:
```
  npm run tw
```
